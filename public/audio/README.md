# Background Music for Lee Roo

Place your ambient music files here for the site to use.

## Required files

- **ambient.mp3** – Main background music (calm, positive, furniture/luxury vibe)
  - Suggested: loopable, 2–5 min, volume-normalized
  - If missing: app uses an online fallback

- **splash.mp3** – Short sound for the splash screen (optional)
  - Suggested: 1–3 seconds, subtle chime or fade-in

## How to add custom music

1. Place `ambient.mp3` in this folder: `public/audio/ambient.mp3`
2. Optionally add `splash.mp3` for the splash screen
3. Restart the dev server if it’s running

## Format

- Preferred: MP3
- Keep file size reasonable (e.g. &lt; 5 MB for ambient)
- Use royalty-free or licensed music
